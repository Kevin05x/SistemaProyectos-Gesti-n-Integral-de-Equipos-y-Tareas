# 💻 SistemaProyectos: Gestión Integral de Equipos y Tareas

> Aplicación web robusta desarrollada con **.NET 8** y **Razor Pages**, diseñada para centralizar la administración de proyectos, coordinación de equipos y seguimiento detallado de actividades.

---

## 📖 Resumen del Proyecto
Esta plataforma optimiza la productividad organizacional mediante un control granular de tareas. Permite a las empresas gestionar múltiples equipos de trabajo, asignar responsables con niveles de prioridad y mantener un flujo de comunicación constante a través de hilos de comentarios en tiempo real.

## 🛠️ Stack Tecnológico

| Capa | Tecnología |
| :--- | :--- |
| **Framework Principal** | .NET 8 (ASP.NET Core Razor Pages) |
| **ORM / Acceso a Datos** | Entity Framework Core |
| **Base de Datos** | SQL Server (Relacional) |
| **Frontend UI** | Bootstrap 5 & FontAwesome |
| **Arquitectura** | Patrón Repository & Inyección de Dependencias |

## 👨‍💻 Contribución y Rol Técnico
En este proyecto participé como **Desarrollador de Software (Colaborador)**, trabajando bajo la dirección de un líder técnico y aportando al ciclo de vida del desarrollo:

* **Implementación de Lógica de Negocio:** Desarrollé módulos clave para la gestión de tareas y sistemas de comentarios, asegurando la integridad de los datos mediante migraciones de **EF Core**.
* **Diseño Colaborativo:** Aporté activamente en sesiones de ideación para optimizar la experiencia de usuario (UX) en los paneles de detalle.
* **Desarrollo Alineado a Estándares:** Implementé código siguiendo los lineamientos de arquitectura definidos por el equipo, facilitando la escalabilidad y el mantenimiento del sistema.
* **Interfaz Responsive:** Aseguré que las vistas detalladas y filtros fueran funcionales en diversos dispositivos utilizando **Bootstrap 5**.

## 🚀 Funcionalidades Clave
* **Control de Acceso (RBAC):** Gestión de permisos basada en roles para proteger la información sensible.
* **Gestión de Equipos:** Asignación dinámica de usuarios a proyectos específicos.
* **Monitoreo de Actividad:** Registro histórico de comentarios y cambios de estado en las tareas.
* **Vistas Avanzadas:** Listados con filtros básicos para una búsqueda rápida de proyectos.
